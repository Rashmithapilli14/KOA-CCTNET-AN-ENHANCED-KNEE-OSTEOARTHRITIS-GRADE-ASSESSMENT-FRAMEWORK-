# KOA-CCTNET-AN-ENHANCED-KNEE-OSTEOARTHRITIS-GRADE-ASSESSMENT-FRAMEWORK-
KOA-CCTNet is an advanced deep learning model for automated knee osteoarthritis (KOA) grade assessment. It uses a modified Compact Convolutional Transformer (CCT) to extract local and global features from X-ray images, improving classification accuracy and supporting early diagnosis.
